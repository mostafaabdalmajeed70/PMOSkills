# PR30 — Monitor and Control Project Work

## Classification
- **Performance Domain:** Measurement
- **Process Group:** Monitoring and Controlling
- **PMBOK 8 Reference:** Section 2.9 (Measurement Performance Domain)
- **Process Groups Practice Guide Reference:** Section 7.1

---

## Purpose
Monitor and Control Project Work is the process of tracking, reviewing, and reporting the overall progress to meet the performance objectives defined in the project management plan. The key benefit is that it allows stakeholders to understand the current state of the project, recognize actions taken to address performance issues, and have visibility into future project status with cost and schedule forecasts.

---

## Inputs
- Project management plan (any component)
- Project documents: assumption log, basis of estimates, cost forecasts, issue log, lessons learned register, milestone list, quality reports, risk register, risk report, schedule forecasts
- Work performance information
- Agreements
- Enterprise environmental factors
- Organizational process assets

---

## Tools & Techniques
- Expert judgment
- Data analysis (alternatives analysis, cost-benefit analysis, earned value analysis, root cause analysis, trend analysis, variance analysis)
- Decision making (voting)
- Meetings

---

## Outputs
- Work performance reports
- Change requests
- Project management plan updates (any component)
- Project documents updates (cost forecasts, issue log, lessons learned register, risk register, schedule forecasts)

---

## Key Concepts
- Monitoring = collecting, measuring, and assessing measurements and trends to effect process improvements.
- Controlling = comparing actual vs. planned performance, analyzing variances, recommending corrective or preventive actions.
- Work performance information is work performance data analyzed in comparison with project management plan components.
- Work performance reports are physical or electronic representations of work performance information circulated to stakeholders.
- This process ensures the project stays aligned with business needs.
