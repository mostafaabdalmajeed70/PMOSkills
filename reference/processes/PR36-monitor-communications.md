# PR36 — Monitor Communications

## Process Identity
| Field | Value |
|---|---|
| **Process ID** | PR36 |
| **Process Name** | Monitor Communications |
| **Process Group** | Monitoring and Controlling |
| **Knowledge Area** | Communications Management |
| **PMBOK Source** | Process Groups: A Practice Guide §7.9 |

---

## Purpose
Ensure the information needs of the project and its stakeholders are met. Optimizes information flow as defined in the communications management plan and the stakeholder engagement plan.

---

## Inputs
| Input | Description |
|---|---|
| Project management plan | Resource management plan, communications management plan, stakeholder engagement plan |
| Project documents | Issue log, lessons learned register, project communications |
| Work performance data | Raw communication effectiveness data |
| Enterprise environmental factors | Organizational culture, communication norms |
| Organizational process assets | Communication templates and policies |

---

## Tools and Techniques
| Tool / Technique | Notes |
|---|---|
| Expert judgment | Evaluating communication effectiveness |
| Project management information system | Communication tracking tools |
| Data representation | Stakeholder engagement assessment matrix |
| Interpersonal and team skills | Observation/conversation |
| Meetings | Status reviews, retrospectives |

---

## Outputs
| Output | Description |
|---|---|
| Work performance information | Assessment of communications effectiveness |
| Change requests | To improve communications plans or activities |
| Project management plan updates | Communications management plan, stakeholder engagement plan |
| Project documents updates | Issue log, lessons learned register, stakeholder register |

---

## Key Concepts
- Determines if communications artifacts have had the **desired effect** on stakeholder support.
- Evaluates right message → right audience → right channel → right time.
- May trigger iteration of Plan Communications Management or Manage Communications processes.
- Methods include surveys, lessons learned observations, issue log reviews, and stakeholder engagement matrix changes.
