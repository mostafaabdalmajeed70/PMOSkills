# C6 — Tools & Techniques Taxonomy
**PMOSkills Repository | PMBOK 8th Edition + Process Groups Practice Guide**
**Version:** 1.0 | **Status:** Complete | **Date:** 2026-05-31

---

## Purpose

This file is the **master taxonomy of all tools and techniques** referenced across the 40 PMBOK processes (reference/processes/). It provides:

- A deduplicated, categorized list of every T&T
- The definition/function of each T&T
- A cross-reference to every process (PR number) where it is used
- A process-count frequency ranking

---

## T&T Category Structure

| # | Category | Description |
|---|---|---|
| 1 | Expert Judgment | Domain knowledge applied to decisions and estimates |
| 2 | Data Gathering | Techniques for collecting information from sources |
| 3 | Data Analysis | Analytical methods for interpreting data |
| 4 | Data Representation | Visual and structural tools for displaying data |
| 5 | Decision Making | Structured methods for selecting among alternatives |
| 6 | Interpersonal & Team Skills | Human-interaction and facilitation techniques |
| 7 | Communication | Methods for information exchange |
| 8 | Estimating | Techniques for quantifying cost, duration, or resources |
| 9 | Planning & Scheduling Tools | Tools for schedule modeling and PMIS |
| 10 | Specialized Techniques | Process-specific techniques not in other categories |
| 11 | Meetings | Structured group interactions |

| T&T | Definition | Processes Using It |
|---|---|---|

| Root Cause Analysis | Identifying underlying causes of variances, defects, or issues | PR19, PR30, PR35 |

| SWOT Analysis | Evaluating Strengths, Weaknesses, Opportunities, and Threats | PR19 |

| Assessment of Other Risk Parameters | Evaluating urgency, proximity, detectability, controllability | PR20 |
| Simulations | Probabilistic modeling (e.g., Monte Carlo) to analyze outcomes | PR21 |
| Sensitivity Analysis | Determining which risks have the most impact on objectives | PR21 |
| Decision Tree Analysis | Modeling decisions with probability-weighted branches | PR21 |
| Influence Diagrams | Graphical models of variables and their causal relationships | PR21 |
| Performance Reviews | Comparing actual performance to planned performance | PR29, PR31, PR34, PR37 |
| Earned Value Analysis (EVA) | Integrating scope, schedule, and cost performance metrics | PR29, PR37, PR40 |
| Trend Analysis | Reviewing project performance over time to forecast future outcomes | PR29, PR37, PR40 |
| Variance Analysis | Comparing planned vs. actual results across performance dimensions | PR29, PR37, PR40 |
| Reserve Analysis | Evaluating adequacy of contingency or management reserves | PR13, PR29, PR34, PR37 |

| Histograms | Bar charts representing frequency distribution of data | PR32 |
| Logical Data Model | Abstract representation of an organization's data | PR03 |

## Category 8 — Estimating

Techniques for calculating effort, duration, cost, or resource quantities.

| T&T | Definition | Processes Using It |
|---|---|---|
| Analogous Estimating | Using historical data from similar projects as the basis for estimates | PR12, PR16 |
| Parametric Estimating | Mathematical models using statistical relationships between variables | PR12, PR16 |
| Bottom-Up Estimating | Aggregating estimates from the lowest level of detail upward | PR12, PR16 |
| Three-Point Estimating | Weighted average of optimistic, most likely, and pessimistic estimates | PR09, PR12 |
| Reserve Analysis (Estimating) | Adding contingency for identified and unidentified risks | PR12, PR13 |
| Cost of Quality (COQ) | Measuring prevention, appraisal, and failure costs in estimating | PR12, PR14 |
