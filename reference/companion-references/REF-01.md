**Shortcode:** APG

**Version:** 1.0.0 · **Status:** Active

## Full Bibliographic Citation
Project Management Institute. (2017). *Agile Practice Guide*. Project Management Institute. Official release: https://www.pmi.org/standards/agile

## What This Reference Adds Beyond PMBOK 8
The Agile Practice Guide provides the operational mechanics of adaptive delivery that PMBOK 8 references but does not fully specify. Where PMBOK 8 establishes adaptive development approach as a valid lifecycle option (§2.3), APG provides Scrum, Kanban, XP, and hybrid implementation guidance. It defines the Agile Manifesto values and principles as operational foundations, and maps agile ceremonies, roles, and artifacts to project governance needs.

## Practitioner Usage Notes
- Consult APG when the project's development approach decision (A13) leans adaptive or hybrid.
- Use APG Appendix X3 for the agile suitability filter when tailoring delivery approach.
- APG Chapter 6 (Implementing Agile) is the key reference for hybrid operating model design.
- For AI agents: load APG mappings when `development_approach: adaptive` or `hybrid` is set in skill YAML.
