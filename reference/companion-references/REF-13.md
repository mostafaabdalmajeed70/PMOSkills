# REF-13 — Project Management Offices: A Practice Guide
**Shortcode:** PMO-Guide  
**Publisher:** Project Management Institute (PMI), 2025  
**Authority:** PMBOK8 Companion — PMO Design, Services, and Value  
**Version:** 1.0.0 · **Status:** Active

## Full Bibliographic Citation
Project Management Institute. (2025). *Project Management Offices: A Practice Guide*. Project Management Institute. Official release: https://www.pmi.org/standards/pmo

## What This Reference Adds Beyond PMBOK 8
The PMO Practice Guide (2025) is the definitive current PMI guidance on PMO design, service catalog development, value measurement, and governance integration. It defines PMO types (Supportive, Controlling, Directive, and Value Delivery), PMO service categories, and the PMO value maturity model. It is the primary reference for all Pack 07 PMO Skills in PMOSkills and for the PMO-layer governance artifacts (A11, A23, A36, A39).

## PMBOK 8 Domain and Principle Coverage

| PMBOK 8 Element | PMO-Guide Contribution |
|---|---|
| Governance Performance Domain (§2.1) | PMO as a governance enabler — service model, authority, and oversight functions |
| P4 — Be an Accountable Leader | PMO accountability: service ownership, value reporting, governance forum facilitation |
| P2 — Focus on Value | PMO value proposition, benefit measurement, and value delivery office model |
| P1 — Adopt a Holistic View | PMO as the integrative layer across multiple projects and portfolios |
