Project Management Institute. (2019). *Practice Standard for Scheduling* (3rd ed.). Project Management Institute. Official release: https://www.pmi.org/standards/scheduling-third-edition

## What This Reference Adds Beyond PMBOK 8
The Scheduling Standard provides the technical specifications for building, maintaining, and analyzing project schedule models — the operational depth that PMBOK 8 references but does not provide. It covers network diagramming conventions, activity sequencing logic types (FS, SS, FF, SF), critical path calculation, schedule compression techniques (fast-tracking, crashing), and schedule model quality criteria. It is the authority reference for any practitioner building a formal schedule model.
