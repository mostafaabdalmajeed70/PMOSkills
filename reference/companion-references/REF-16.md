# REF-16 — The Standard for Earned Value Management
**Shortcode:** EVM

## Full Bibliographic Citation
Project Management Institute. (2020). *The Standard for Earned Value Management*. Project Management Institute. Official release: https://www.pmi.org/standards/earned-value-management

## What This Reference Adds Beyond PMBOK 8
EVM Standard provides the full technical specification for integrated cost-schedule performance measurement using Earned Value Analysis. While PMBOK 8 references EVM metrics (CV, SV, CPI, SPI, EAC, VAC) as monitoring and control tools, this standard defines the Performance Measurement Baseline (PMB) construction, work authorization procedures, and the formal Integrated Baseline Review (IBR) process. It is essential for any project requiring formal EVM compliance (defense, government, large infrastructure).

| P3 — Embed Quality | EVM data quality: accurate timephasing, work authorization, and progress measurement |
