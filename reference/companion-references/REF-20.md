## What This Reference Adds Beyond PMBOK 8
This guide defines organizational change management (OCM) as a practice that integrates with — but is distinct from — project management. It provides the change readiness assessment model, stakeholder impact analysis, adoption planning, and resistance management techniques that support project benefit realization. PMBOK 8 covers change readiness as an artifact area (A29) but does not provide OCM methodology depth; this guide fills that gap.
