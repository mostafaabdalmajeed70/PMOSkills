# REF-14 — The Standard for Portfolio Management (4th ed.)
**Shortcode:** Portfolio-4E  
**Publisher:** Project Management Institute (PMI), 2017

**Version:** 1.0.0 · **Status:** Active

## Full Bibliographic Citation
Project Management Institute. (2017). *The Standard for Portfolio Management* (4th ed.). Project Management Institute. Official release: https://www.pmi.org/standards/for-portfolio-management

## What This Reference Adds Beyond PMBOK 8
The Portfolio Management Standard defines the processes and governance model for managing a portfolio of projects and programs to achieve strategic objectives. It provides the intake, prioritization, balancing, and performance monitoring framework that sits above the individual project level. For PMOSkills, it is the authority reference for portfolio governance artifacts (A10, A22) and for understanding how project governance fits within a portfolio governance system.

## PMBOK 8 Domain and Principle Coverage

| PMBOK 8 Element | Portfolio-4E Contribution |
|---|---|
| Governance Performance Domain (§2.1) | Portfolio governance: intake, authorization, performance monitoring, strategic alignment |
| P2 — Focus on Value | Portfolio value optimization: balancing investment across the portfolio for maximum strategic return |
| P1 — Adopt a Holistic View | Portfolio as a system: interdependencies, shared resources, and strategic coherence |
