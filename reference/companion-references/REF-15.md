## What This Reference Adds Beyond PMBOK 8
The Program Management Standard defines program governance, benefits management, stakeholder engagement, and organizational change management at the program level — the tier between individual projects and portfolio management. It is particularly relevant for PMOSkills practitioners managing interconnected projects whose combined outcomes deliver benefits that no single project could achieve alone. It defines the benefits realization governance model that informs A02 and A24.
