## What This Reference Adds Beyond PMBOK 8
This guide provides the implementation roadmap for standing up OPM capability in an organization — covering maturity assessment, gap analysis, implementation planning, and change management for OPM adoption. Where the OPM Standard (REF-12) defines what OPM is, this guide explains how to build it. It is the primary reference for PMO Directors undertaking a PMO setup or capability expansion initiative.
