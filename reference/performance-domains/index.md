# Performance Domains Index — PMBOK 8 Canonical
**Type:** Reference Index  
**Edition:** PMBOK 8th Edition (2025)  
**PMBOK 8 Source:** PMBOK8 Guide Part 2 §2.1–§2.7 — Performance Domains  
**Version:** 2.0.0  

## Purpose
Performance domains are the **seven** interactive, interrelated, and interdependent areas of focus that work in unison to achieve the desired project results (Guide §2). PMBOK 8 reorganizes the Guide around these seven domains, replacing the ten Knowledge Areas from PMBOK 6 and the eight domains from PMBOK 7.

Performance domains are not sequential phases; they represent concurrent areas of focus that operate simultaneously from initiation through closure.

## Domain Interaction and Integration
- **Governance** serves as the central coordinating mechanism that monitors scope, schedule, and financial trade-offs to maximize value delivery.
- **Scope, Schedule, and Finance** are the core delivery baselines, where changes in one systemically impact the others.
- **Stakeholders and Resources** drive execution; human interactions, skills, and effective communication enable WBS tasks.
- **Risk** spans all domains, identifying threats to baselines and opportunities to optimize resources or schedule speed.

## PMBOK 7 Heritage Reference (Cross-Edition Traceability)
> PMBOK 7 utilized eight performance domains (Stakeholders, Team, Dev Approach, Planning, Project Work, Delivery, Measurement, Uncertainty). PMBOK 8 synthesizes and restructures these into its seven canonical domains. Historical PMBOK 7 files have been moved to the `pmbok7-archive/` directory.

| PMBOK 7 Performance Domain | Primary Destination in PMBOK 8 |
|---|---|
| Stakeholders | PD8-05 — Stakeholders Performance Domain |
| Team | PD8-06 — Resources Performance Domain (Lead the Team) |
| Development Approach and Life Cycle | PD8-01 — Governance / PD8-02 — Scope / PD8-03 — Schedule |
| Planning | Fully integrated across all domains (PD8-01 through PD8-07) |
| Project Work | PD8-01 — Governance / PD8-06 — Resources / PD8-05 — Stakeholders |
| Delivery | PD8-02 — Scope Performance Domain (including quality) |
| Measurement | PD8-01 — Governance (Monitor and Control Project Performance) / EVM |
| Uncertainty | PD8-07 — Risk Performance Domain |
