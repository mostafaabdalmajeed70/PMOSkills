## Purpose

This checklist verifies that performance monitoring metrics, issue tracking systems, change control paths, and ESG data capture are configured and functioning correctly to support active steering.

---

## Readiness Checklist

| Category | Check Item | Target Skill / Reference | Verified Status |
|---|---|---|---|
| **Tracking** | [ ] Schedule tracking models and Earned Value formulas (A15) are active. | `SKL-05-02` / `A15` | [ ] Yes / [ ] No |
| **Tracking** | [ ] Cost tracking system (A16) is linked to accounting and billing codes. | `SKL-05-03` / `A16` | [ ] Yes / [ ] No |
| **Changes** | [ ] Integrated Change Control procedures and `A12Change Log` are active. | `SKL-05-08` / `A12` | [ ] Yes / [ ] No |
| **Issues** | [ ] Issue and Action Record (A18) is set up with tracking owners assigned. | `A18` | [ ] Yes / [ ] No |
| **ESG** | [ ] Sustainability tracking parameters and ESG reporting models are configured. | `A-NEW-SUST` | [ ] Yes / [ ] No |
| **Routines** | [ ] Monthly performance reporting templates (A17) are established. | `A17` | [ ] Yes / [ ] No |

---

## Gate Transition Criteria

*   **PASS (6 / 6 met):** Steering tools are online. Ready for active tracking.
*   **WARN (5 / 6 met):** ESG metrics are not yet connected to central database, but manual tracking is configured. **Condition:** Action plan to automate in 7 days.
*   **FAIL (< 5 met OR Change Log inactive):** Cannot proceed. Uncontrolled variances cannot be steering without functioning change control.
