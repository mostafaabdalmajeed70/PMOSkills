## Purpose

This section pattern defines a standard markdown template layout for documenting outputs of a process or skill file, ensuring standard tracking of deliverables.

---

## The Outputs Markdown Template

```markdown
## Outputs

Primary deliverables generated by executing this skill:

| Output ID | Output Artifact Name | Target Path | Approval Authority | Reference Link |
|---|---|---|---|---|
| A08 | Scope & Requirements Record | `artifacts/planning/` | Sponsor or CCB | [A08](../../artifacts/planning-and-baselines/A08-scope-requirements-backlog.md) |
| A-NEW-SUST | Sustainability Register | `artifacts/planning/` | Sponsor / PMO | [A-NEW-SUST](../../artifacts/planning-and-baselines/A-NEW-SUST-sustainability-register.md) |
```
